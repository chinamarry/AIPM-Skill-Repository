#HTML转PRD技能

自定义AIPM技能合集——专为AI PM打造的工具链。

---

##技能列表

###html转prd

>Axure 高保真原型HTML → AI编程专用PRD html-to-prdrkdown文档生成器

**核心功能**：从 Axure 导出的高保真 HTML 原型文件中，自动提取所有页面元素、动态面板、中继器控件、二级级联面板、列表字段、筛选条件、统计指标、页面弹窗、Tab 页结构、导航层级关系和交互动作等，生成结构化的 PRD 文档。

**方法论亮点**（26 节，覆盖 39 类典型错误）：

| 节号 | 方法论名称               | 解决的问题                     |
| ---- | ------------------------ | ------------------------------ |
| 1    | 基础扫描流程             | 建立标准化的 HTML 扫描工作流   |
| 2    | 文件编码检测             | 处理 GB2312/GBK 编码文件       |
| 3    | 文件编码转换             | 自动转换编码避免乱码           |
| 4    | 字段坐标位置分析法       | 区分筛选条件、表格列、弹窗区域 |
| 5    | 字段提取与映射规则       | 字段编号、命名规范             |
| 6    | 约束条件提取规则         | 从 HTML 注释中提取字段校验规则 |
| 7    | 下拉框完整扫描           | 枚举值、二级级联完整提取       |
| 8    | 分组标题识别             | 识别分割线、二级/三级标题      |
| 9    |选项卡页结构完整识别|识别动态面板状态结构|
| 10   | 导航层级关系图生成规则   | Tab 内弹窗归属、详情页命名规范 |
| 11   |选项卡内按钮识别规则|扫描选项卡页内所有操作按钮|
| 12   | 删除操作识别规则         | 搜索删除按钮及确认提示框       |
| 13   | 单选交互说明规则         | 详细描述单选交互规则           |
| 14   | 返回按钮识别规则         | 识别详情页标题区域的返回按钮   |

| 16   | 字段排列顺序规则         | 按 data-left 坐标排序          |
| 17   | 二级级联完整扫描         | 搜索"二级级联"关键词           |
